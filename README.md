## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Avesh6754/space_study_app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd space_study_app
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
